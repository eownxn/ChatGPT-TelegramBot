
# ChatGPT in Telegram by aiogram v3.x.x
### 

## Usage/Examples

$ pip install -r requirements.txt

Create file with name config.py and put there:
```python
BOT_TOKEN = 'your telegram bot token here'
CHATGPT_TOKEN = 'openai token here'
```

$ python main.py
